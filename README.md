# NVmesh

NVmesh is a specialized NVLink scale-up interconnect measurement toolkit extended from [NVIDIA nvloom](https://github.com/NVIDIA/nvloom).

This repository hosts the measurement tool developed for the SIGCOMM 2026 paper *Scale-Up Networks for Large Language Models: Measurement, Implications and Optimization*.

NVmesh is designed to characterize scale-up GPU interconnects in modern LLM systems, enabling fine-grained measurement, profiling, and analysis of NVLink behavior.

The codebase is currently being organized and will be open-sourced soon.
